I. บทที่ 12.7   KeyboardEvent and PreventDefault


keyup keydown กดลงขึ้น keydown พอปล่อยขึ้น keyup กดลงค้างขึ้น keydown เเล้วนับจำนวนครั้ง

EX ในส่วนของ JS
const input = document.querySelector('input')

input.addEventListener("keydown", function () {
  console.log("keydown");
})

input.addEventListener("keyup", function () {
  console.log("keyup");
})

EX2 กด key เเล้ว ออกมาเป็นอะไร เช่น กด K เเสดง เป็น กด K ออก เป็นปุ่ม K 
input.addEventListener("keydown", function (e) {
  console.log(e.key);
  console.log(e.code);
})


EX3 เเสดงซ้อยขวา
const input = document.querySelector('input')

input.addEventListener("keydown", function (e) {
  // console.log(e.key);
  if (e.key == "ArrowUp") {
    console.log("Up");
  } else if (e.key == "ArrowDown") {
    console.log("Down");
  }
  if (e.key == "ArrowLeft") {
    console.log("Left");
  }
  if (e.key == "ArrowRight") {
    console.log("Right");
  }
})
