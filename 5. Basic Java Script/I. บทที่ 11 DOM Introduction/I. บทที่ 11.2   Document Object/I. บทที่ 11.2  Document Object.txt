I. บทที่ 11.2  Document Object


DOM เริ่มจาก              	-> LI
Document -> body ->  UL	-> LI
	         ->  H1


-Document เปรียบเสมือน ประตูเข้าสู่ DOM กรอบ กรอบใหญ่ ที่เข้าหน้า web page 
ไปเช็คใน console เเต่ละเพจได้เลย 
พิมพ์ Document หรือจะดูชนิดทั้งหมเดที่ใช้ พิมพ์ Document.all 
เสร็จเเล้วเวลาจะเข้าถึงตำเเหน่งหรือบรรทัดนั้นๆ ก็พิมพ์  Document.all[index]
วิธีการเข้าถึงจะเป็นเเบบ index เเต่ไม่ใช่ Array นะ
EX
Document.all[10].innerHTML(หมายถึงการเเก้ไข) = "SILKIE" ทำเเบบนี้ทำให้เราสามารถเเก้ไขได้โดยไม่ต้องเข้า HTML
