JavaScript 1: พื้นฐาน
เข้าสู่โลกของการพัฒนาเว็บไซต์ด้วยหลักสูตร JavaScript 
ระดับพื้นฐานที่ออกแบบมาสำหรับผู้เริ่มต้นและผู้ที่ต้องการเสริมความเข้าใจในแนวคิดพื้นฐานของ 
JavaScript เราจะเริ่มต้นด้วยการแนะนำถึงตัวแปรและค่าต่างๆ ที่สามารถเก็บไว้ในตัวแปรเหล่านั้น 
รวมทั้งการประกาศตัวแปรโดยใช้ var, let, และ const เพื่อให้เข้าใจถึงขอบเขตและกรณีการใช้งานของแต่ละประเภท 
เราจะสำรวจวิธีการอัปเดตค่าในตัวแปรและการจัดการกับค่าว่าง เช่น null และ undefined นอกจากนี้ 
เราจะศึกษาการใช้งานเครื่องหมายเปรียบเทียบ และสำรวจโครงสร้างการควบคุมโฟลว์ของโปรแกรม เช่น คำสั่ง else-if 
และคำสั่ง switch-case สุดท้ายนี้ เราจะแนะนำการใช้งานคอนโซลในการดีบักและทดสอบโค้ด JavaScript ของคุณ
