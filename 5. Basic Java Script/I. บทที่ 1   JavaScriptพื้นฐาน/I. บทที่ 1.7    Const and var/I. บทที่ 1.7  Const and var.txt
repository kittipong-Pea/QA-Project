I. บทที่ 1.7  Const and var

การประกาศตัวเเปล

Const คล้ายๆ let เเต่ไม่สามารถ เเก้ไข ต่อได้
ทำไมถึงใช้ การเก็บค่าที่คงที่ ไม่เปลี่ยนเเปลง เช่นวัน ยังไงก็ต้องมี 7วัน
const name = values ;

var
เป็นการประกาศ ที่เก่ามาก ทำงานเหมือน let 
Var กับ Let ต่างกันยังไง

Var เป็น gobal scope ได้ทั้งหมดเลย 
Let เป็น  block scope ได้เเค่อยู่ใน scope นั้นเท่านั้น

var name = values ;
