JavaScript 3: Functions
สำรวจโลกแห่งฟังก์ชันใน JavaScript เรียนรู้การใช้งานฟังก์ชันใน JavaScript อย่างลึกซึ้ง 
หลักสูตรนี้ได้ถูกออกแบบมาเพื่อผู้ที่มีพื้นฐาน JavaScript และต้องการขยายความรู้ในด้านฟังก์ชัน 
ซึ่งเป็นหนึ่งในแนวคิดหลักที่จำเป็นสำหรับการพัฒนาเว็บไซต์และแอปพลิเคชั่น เราจะเริ่มต้นด้วยพื้นฐานของตัวแปรและค่าต่างๆ, 
การประกาศ, การอัปเดตตัวแปร, และการเข้าใจประเภทข้อมูลพื้นฐานรวมถึงค่าว่าง (null, undefined)
