I. บทที่ 3.13 Logical Not


not เปลี่ยน จาก  true เป็น false เเละ false เป็น true ได้

เคื่องหมาย   !

8<9  true
!8<9 false

Ex

let username = prompt("pls enter username")

if (!username){
    console.log("true");

}else{
    console.log("false");
}

EX2

let a = 9
let b = 8
if(!a < b){
    console.log("a < b");
}else{
    console.log("a > b");
}
