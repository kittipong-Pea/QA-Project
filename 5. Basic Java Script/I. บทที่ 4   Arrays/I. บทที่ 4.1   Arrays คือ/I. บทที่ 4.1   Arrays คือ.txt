I. บทที่ 4.1   Arrays คือ

Arrays เป็น Data Structure

Data Structure (โครงสร้างข้อมูล) คือ “วิธีการจัดเก็บและจัดระเบียบข้อมูลในคอมพิวเตอร์” เพื่อให้สามารถเข้าถึง แก้ไข และใช้งานได้อย่างมีประสิทธิภาพ

Arrays คือ collection ของ values
เก็บข้อมูลเป็นช่อง ๆ ขนาดเท่ากัน
เข้าถึงเร็ว แต่เพิ่ม/ลบยาก 
Linked List
ข้อมูลเชื่อมกันเป็นโหนด
เพิ่ม/ลบง่าย แต่เข้าถึงช้า
Stack (LIFO: เข้า-ออกท้าย)
เหมือนกองจาน
ใช้ใน: undo, call stack
Queue (FIFO: เข้า-หน้า ออก-หลัง)
เหมือนการต่อคิว
ใช้ใน: งานระบบ, task queue


กลับมาที่ array 
ใช้ [] ในการเก็บ สามารถเก็บได้หลาย type เลย
[] ;

EX
let name = [123,114,"abc"]

ถ้าใช้  length
name.length จะเเสดงออกมาเป็น 1 คำนั้นๆ เเทนที่จะเป็น 1 ตัวอักษร
หรือใช้ index ก็เริ่อมจาก 0 เหมือนเดิม
name[0] ตัวที่1
