I. บทที่ 4.7   Equality

reference type

let num1 =[1,2,3]
let num2 =[1,2,3]

num1 == num2 
false	มันจะมองว่าไม่เท่ากัน

เเต่ถ้าเรา
let num3 = num2
num3 == num2
true	มันจะมองว่าเท่ากันเพราะ เราตั้งค่า ให้มัน reference ให้มันเหมือนกันเรียบร้อยเเล้ว
**มีข้อเสียอย่า เวลาเราเเก้ไขเพิ่มเติม array ไหนก็ตาม มันก็จะเเก้ไปด้วย**
