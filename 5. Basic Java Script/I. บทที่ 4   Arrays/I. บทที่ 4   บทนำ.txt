JavaScript 2: Object, Arrays และ Loops
หลักสูตรถูกออกแบบมาเพื่อให้ผู้เรียนเข้าใจและสามารถใช้งาน 
object (ออบเจกต์), arrays (อาร์เรย์) และ loops (ลูป) ใน JavaScript 
ได้อย่างคล่องแคล่ว ซึ่งเป็นพื้นฐานสำคัญในการพัฒนาแอปพลิเคชันและเว็บไซต์ที่มีประสิทธิภาพ ในหลักสูตรนี้ 
ผู้เรียนจะได้รับความรู้เกี่ยวกับการสร้างและการจัดการ array เพื่อเก็บข้อมูลแบบลำดับ การใช้งาน object 
เพื่อเก็บข้อมูลแบบมีโครงสร้าง และการใช้ loop เพื่อทำซ้ำกระบวนการหรือการเข้าถึงข้อมูลในออบเจกต์และอาร์เรย์
