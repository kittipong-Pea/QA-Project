I. บทที่ 4.8   การใช้ consts กับ array


ทำไมนิยมใช้ const กับ array 

เราสามารถ เพิ่มเเก้ไขได้ เเต่เเก้ไข array ทั้งก้อนไม่ได้ การ add ค่ากลับไป ไม่ได้
EX
const name =['1','2'];
name.push('3');
name[0] = '4';
เเบบพวกนี้ได้

เเต่ เเบบนี้ไม่ได้ การ add ค่ากลับไป ไม่ได้
name =['5','6'];


เอาไว้กันค่าที่  add ค่ากลับไป ไม่ได้
